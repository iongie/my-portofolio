Point Of Sales
Ini merupakan source kode user interface dan API dari aplikasi Point of Sales.
Teknologi yang digunakan
Dalam pembuatannya menggunakan Angular sebagai framework javascript. Dengan ditunjang Coreui untuk angular sebagai template dashboar admin yang didukung teknologi Bootstrap serta Angular Material sebagai teknologinya. Dengan Menggunakan Mysql sebagai solusi penyimpanan dan socket.io sebagai protokol komunikasi data.
